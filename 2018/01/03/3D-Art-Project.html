    3D Art Project

    January 2018

    Who said art and technology don’t go together? I designed and built a holographic art piece using a laser-engraved piece of acrylic, RGB LED strip, and custom 3D-printed base.

    Documentation and Log

    Introduction

    The goal of this project is to create an engraved art piece on acrylic and use an RGB LED strip to project unique light patterns into it from underneath. To accomplish this, the final product will consist of a 3D-printed base that houses the electronics and LED strip as well as a piece of acrylic that has been cut and engraved using a laser engraver to create an artistic design.

    This was a project done during my high school digital electronics class.

    Day 1

    The assembly (soldering) of the internal electronics and LED strip was completed. The internal electronics include an Adafruit Trinket and one resistor soldered to a piece of perfboard as well as the RGB LED strip, which is connected to the Trinket using three wires (VCC, GND, and data).

    Day 2

    I tested the internal electronics, including the LEDs, with example code from the FastLED library. Additionally, I started the 3D model of the base that will house the electronics in Autodesk Inventor.

    Day 3

    I soldered the USB extension cable so that the switch can be used to toggle power to the device and the USB cable can be disconnected without opening the base. The female USB micro-B connector caused issues because the pins were minuscule and quite difficult to solder. I overcame the problem using flux and thinner wire (26 gauge instead of 22). Also, I began printing the 3D model of the base.

    Day 4

    I cleaned and assembled the base, placing all the components inside and making necessary tweaks. Everything seemed to work properly…

    Day 5

    After testing the electronics, I ran into a problem: the Trinket would no longer accept data and couldn’t be programmed. The LEDs began to flicker as the cables from the female USB connector appeared to be coming loose. X( I ended up desoldering and replacing the Trinket, which had stopped working likely due to a short in the USB connector. In place of the female USB connector, I decided to use a female USB breakout board, which would prevent the cables from shorting and damaging the new Trinket. The 3D model was modified accordingly to fit the new breakout board.

    Day 6

    I created the design (in Adobe Illustrator) that would be used to cut and engrave a design on the piece of acrylic.

    Day 7

    I replaced the old USB connectors with the USB breakout board. Since the 3D-printed base is easy to open, I decided to connect only VCC and GND from the breakout board (skipping the USB data wires) and open the base to reprogram the Trinket when necessary. Also, I began printing the new 3D model. Finally, I programmed the Trinket to display several animations in a repeating cycle.

    Here’s the code:

    #include <FastLED.h> //import FastLED library

#define LED_PIN     1 /* pin used for the LED data connection - mine is soldered to pin 1 */
#define NUM_LEDS    15 /* length of LED strip - mine is 15 LEDs */
#define BRIGHTNESS  255 /* maximum brightness */
#define LED_TYPE    WS2812 /* type of LED chip, FastLED needs to know how to communicate with these */
#define COLOR_ORDER GRB /* order of the color values, also needed by FastLED for this LED strip */
CRGB leds[NUM_LEDS]; /* creates a CRGB object, used to store and manipulate the LED colors */

void setup() {
       delay(1000); /* wait 1 sec - helps prevent damage to the LED strip if cable is plugged in slowly/erratically */
       FastLED.addLeds<LED_TYPE, LED_PIN, COLOR_ORDER>(leds, NUM_LEDS).setCorrection(TypicalLEDStrip); /* initiate FastLED with values defined above */
       FastLED.setBrightness(BRIGHTNESS); /* set brightness as defined above */
}

void loop() {
       /* begin animations */

       /* Animation 1 */
       for (int i = 0; i <= 510; ++i) { /* loop this using 'i' values of 0 to 510, limits how long this runs */
              static int hue = 0; /* variable that stores the hue value  */
              FastLED.showColor(CHSV(hue++, 255, 255)); /* set all the LEDs to the current hue, then increment the hue by 1 */
              delay(20); /* wait 20ms */
       }
       /* This animation creates a slow color change across the entire spectrum of RGB colors */

       /* Animation 2 */
       for (int c = 0; c < 3; ++c) { /* loop this three times */
              int brightness = 0; /* create a brightness variable, zero initially */
              for (int i = 0; i < BRIGHTNESS; ++i) { /* repeat BRIGHTNESS times, respecting the BRIGHTNESS option set at the beginning */
                     FastLED.showColor(CHSV(160, 255, brightness++)); /* show a blue color with the current brightness, then increase the brightness by 1 */
                     delay(7); /* wait 7ms */
              }
              for (int i = 0; i < BRIGHTNESS; ++i) { /* repeat BRIGHTNESS times, respecting the BRIGHTNESS option set at the beginning */
                     FastLED.showColor(CHSV(160, 255, brightness--)); /* show a blue color with the current brightness, then decrease the brightness by 1 */
                     delay(7); /* wait 7ms */
              }
       }
       /* This animation creates a blue glow that fades from minimum to maximum brightness three times */

       /* Animation 3 */
       for (int n = 0; n < NUM_LEDS; ++n) { /* repeat for all LEDs (n=0 to n=14 in this case) */
              leds[n] = CHSV(map(n, 0, (NUM_LEDS - 1), 0, (255 - (255 / NUM_LEDS))), 255, 255); /* set the hue of the n'th LED, which should be the LED's position, 0 to NUM_LEDS, scaled to a value 0 to 255. This gives a roughly equally distributed spectrum of colors. */
       }
       for (int c = 0; c < 40; ++c) { /* cycle this 40 times */
              FastLED.show(); /* display the current spectrum of colors */
              CRGB temp = leds[0]; /* store the color of the zero'th LED in a temporary variable */
              for (int i = 0; i < NUM_LEDS; ++i) { /* repeat for each LED */
                     if (i == (NUM_LEDS - 1)) { /* if we're on the last LED, set its color to the color of the first LED (looping) */
                           leds[i] = temp;
                     }
                     else { /* otherwise, move the current LED one spot over */
                           leds[i] = leds[i + 1];
                     }
                     delay(20); /* wait 20ms */
              }
       }
       /* This animation creates the rainbow spectrum across the 15 LEDs, moving it across the strip LED by LED and looping the end back to the beginning*/

       /* end of animations, loops back to the beginning */
}

    Day 8

    I inspected the printed base and acrylic and assembled the entire project. Everything worked!
